# 🎲 Projeto Sorteador

Este repositório contém o projeto **Sorteador**, desenvolvido como parte do curso:

**Git e GitHub: compartilhando e colaborando em projetos** da Alura.

## 📚 Objetivo

O principal objetivo deste projeto foi aplicar, na prática, conceitos fundamentais de versionamento de código utilizando Git e GitHub, além de reforçar boas práticas de colaboração em projetos.

## 🚀 Conteúdos revisados

Durante o desenvolvimento deste projeto, foram revisados os seguintes tópicos:

- Criação de repositórios Git
- Compartilhamento de projetos:
  - Via terminal
  - Via IDE
- Gerenciamento de commits:
  - Desfazer alterações
  - Resetar commits
  - Remover commits do histórico
- Uso do `.gitignore` para ignorar arquivos desnecessários
- Compartilhamento de trechos de código com **Gists**
- Navegação no histórico:
  - Como voltar para commits anteriores
- Configuração de autenticação com **chaves SSH**
- Resolução de conflitos de merge

## 🛠️ Tecnologias utilizadas

- Git
- GitHub

## 📌 Observações

Este projeto tem fins educacionais e foi desenvolvido como parte do processo de aprendizado em controle de versão e colaboração em equipe.
